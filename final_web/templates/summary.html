Summary

Predicting Song Genre Using Machine Learning Methods

Can lyrics and sound features predict song genre?

Our team was curious about the relationship between song genre, song lyrics, and sound features. We aimed to use various machine learning models to predict song genre using lyrics and sound data. Below are some detailed explanations of the machine learning methods involved and our findings. You can find the project on Github here.

The Data

The datasets used for this project were obtained from www.spotify.com and www.azlyrics.com . The initial dataset gathered from spotify api contains 114,832 songs from 3,132 artists and 111 song genres while the azlyrics dataset contains 147,872 songs from 6,464 artists and 111 song genres.These datasets include songs in different languages. After the cleaning process, the datasets from both sources were reduced to top 7 genres and only include songs in English.

Data Cleaning

There were variety of issues with the gathered datasets considered before the cleaning process in order to determine which features and values need to be extracted or removed. Some of the limitations of our datasets are expalained below.

Spotify

The data cleaning process for the spotify dataset includes:

  • Genre Selection - Spotify API Request
  • Track Information - Spotify API Request
  • Audio Features - Spotify API Request

AZLyrics

The data cleaning process for the AZLyrics dataset includes:

  • Drop duplicates and NA values.
  • Standardize artist names to match names from Spotify dataset.
  • Remove songs with invalid and non-english lyrics.
  • Merge lyric dataset with Spotify dataset on artist and song name to obtain genre.
  • After the merge, only about 7,000 songs remain with 90 genres.
  • Stopwords retained for more accurate model after testing.

The Models

Spotify

  • Audio Features - Neural Network Deep Learning - Accuracy 0.26 - Baseline Accuracy 0.01
  • Audio Features - Neural Network Deep Learning - Accuracy 0.57 - Baseline Accuracy 0.14
  • Audio Features - KNN Model - Accuracy 0.55 - Baseline Accuracy 0.14

AZLyrics

  • Data Not Normalized - Take Top 30 Most Popular Genres to Improve Accuracy
  • Multiple Models Tested for Natural Language Processing
  • Lyrics - LinearSVC for Natural Language Processing - Accuracy 0.36 - Baseline Accuracy 0.06

Future Improvements

Our models' accuracy would be greatly improved with a larger dataset with fewer categories. We would like to try improving prediction accuracy by building a model that uses both the lyrics and audio features of the songs to predict the genre. A few paths forward with this data are (1) Artificial Intelligence that is trained to write lyrics based on a genre and (2) Perform sentiment analysis based on more features from the Spotify API.
